# HTML Forms Practice

This project is a beginner-friendly HTML form created to practice different form elements and understand how user input is collected on web pages.

## Features

* Email input field
* Date/Time input
* Dropdown menu
* Textarea
* Radio buttons
* Submit button
* Fieldset and legend for better form organization

## Built With

* HTML5
* CSS3 (if styling is added)

## What I Learned

While building this project, I practiced:

* Creating forms in HTML
* Using labels and input fields
* Working with dropdown menus
* Using radio buttons and textareas
* Organizing forms with fieldsets
* Basic webpage structure

## Future Improvements

* Add form validation
* Improve styling with CSS

* Make the form responsive
